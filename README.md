## Project Overview
This project presents an interactive IMDb Movies Dashboard built using Tableau.
The dashboard analyzes movie ratings, genres, gross revenue, votes, and release trends.

## Key Features
- KPI analysis: IMDb Rating, Meta Score, Votes, Gross Revenue, Duration
- Top 10 movies by number of votes
- Genre-wise IMDb and Meta Score comparison using box plots
- Movie trends by release year
- Gross performance analysis over time

## Tools Used
- Tableau Public
- Microsoft Excel / CSV

## Files Included
- Tableau Dashboard (.twbx)
- Dataset (.csv)
- Dashboard screenshots
